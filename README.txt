=== XAUUSD SUPREME TRADING BOT ===

SETUP:
1. Install Python 3.12+ from python.org
2. Open CMD and run:
   pip install MetaTrader5 pandas numpy requests
3. Install Ollama from ollama.com
4. Open CMD and run:
   ollama pull hermes3:3b
5. Create Telegram bot:
   - Message @BotFather on Telegram
   - Send /newbot
   - Copy the token
6. Get your chat ID:
   - Message @userinfobot on Telegram
   - Copy your ID
7. Open supreme_bot.py in Notepad
8. Replace:
   TELEGRAM_TOKEN = "YOUR_TELEGRAM_BOT_TOKEN_HERE"
   TELEGRAM_CHAT_ID = "YOUR_CHAT_ID_HERE"
9. Open MetaTrader 5, login to demo account
10. Run: python supreme_bot.py
